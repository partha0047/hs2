# Images Directory

Place your property images in this directory. The following images are referenced in the site:

## Required Images

### Hero & Main Images
- `hero.jpg` - Main hero image (interior photo, recommended: 1920x1080)
- `premium-suite.jpg` - Premium suite hero image (recommended: 1200x800)

### Suite Images
- `suite-2.jpg` - Special suite image (recommended: 800x1000)
- `suite-3.jpg` - Couples suite image (recommended: 800x1000)

### Gallery Images
- `gallery-1.jpg` - Living room photo (recommended: 800x800)
- `gallery-2.jpg` - Bathroom photo (recommended: 800x800)
- `gallery-3.jpg` - Minimalist corner (recommended: 800x800)
- `gallery-4.jpg` - Modern bathroom sink (recommended: 800x800)
- `gallery-5.jpg` - Bright room with natural light (recommended: 800x800)

### Testimonial
- `testimonial-1.jpg` - Testimonial avatar (recommended: 400x400, circular crop)

## Placeholder Option

For development, you can use placeholder services like:
- `https://placehold.co/1920x1080/F4F0E8/3F4A43?text=Hero+Image`
- Or use `https://picsum.photos/1920/1080` for random images

Update the image paths in `src/content/site.ts` if using external URLs.
